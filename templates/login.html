¿Dónde puedo obtener más información?

La documentación de 2i2c Hubs tiene toda la información que necesita para usar el hub, así como información sobre la tecnología de implementación.

¿Qué es un Jupyter Notebook?

Jupyter Notebook es una aplicación web de código abierto que le permite crear y compartir documentos que contienen código en vivo, ecuaciones, visualizaciones y texto narrativo. Entre sus usos se incluyen: limpieza y transformación de los datos, simulaciones numéricas, modelado estadístico, visualización de los datos, aprendizaje automático y mucho más.

¿Qué es JupyterHub?

JupyterHub proporciona el poder de las notebooks a grupos de personas usuarias. Permite a las personas usuarias acceder a entornos y recursos informáticos sin abrumarlas con tareas de instalación y mantenimiento. Las personas usuarias, incluidas estudiantes, investigadoras y científicas de datos, pueden hacer su trabajo en sus propios espacios con recursos compartidos que quien administra el sistema puede gestionar eficientemente.

¿Quién puede utilizar este servicio?

El servicio de JupyterHub está abierto para cualquier persona que un administrador del hub haya autorizado. Por lo general, se trata de docentes y estudiantes. Dado que este es un servicio de prueba de concepto, el estudiantado debe asegurarse de hacer una copia de seguridad de sus notebooks regularmente si utilizan este servicio para las actividades del curso.

Este servicio se ejecuta de manera transparente desde github.com/2i2c-org/infrastructure

Diseñado con Kubernetes, Helm, Docker, Miniforge y muchas partes del proyecto Jupyter
